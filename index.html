NoScroll.js

NoScroll.js is a library that prohibits scrolling.
Let's try scrolling the box below!

Couldn't you scroll?
Now let's scroll out of the box!
You should be able to scroll.

Easy to use. Instantiate the NoScroll constructor after loading the NoScroll.js file.

var noscroll = new NoScroll();

Then add the noscroll class to the element you want to prohibit scrolling.

<div class ="noscroll">

Multiple scroll-prohibited elements can be generated.
It takes away the scrolling function of all devices, including PCs, tablets and smartphones!
